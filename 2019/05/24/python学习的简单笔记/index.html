执行环境

compile()

compile()函数允许程序员在运行时刻快速生成代码对象，然后可以使用exec和eval()来执行这些代码对象或求值。
eval()

exec

eval()

eval() 对表达式求值

exec

exec 和 eval() 类似，执行代码对象或字符串形式的python代码。

input()

input() 的作用和 eval(raw_input()) 一样，会将用户输入的内容进行求值然后返回表达式结果。
对raw_input() 和 input() 的输入结果进行一个比较：

execfile()

同exec一样读取文件里面的代码并执行

python -c 命令行

可以直接执行python代码
不需要进入python交互式界面，直接在终端运行python代码

os.system()

接受字符串形式的系统命令并执行它

os.popen()

popen() 是文件对象和system()函数的结合，工作方式和system()类似，但是可以建立一个指向那个程序的单向连接，然后像访问文件一样访问这个程序。

popen()返回的是一个类文件对象。

sys.exit()

退出python程序
