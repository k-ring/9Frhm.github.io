正则(re)的初窥门径

发表于 2019-06-07 | 分类于 python
字数统计: 1,900 字 | 阅读时长 ≈ 8 分钟
文字处理，网页填表，数据抓取，新闻列表，数据清洗等，这些都需要正则，正则表达式为高级的文本模式匹配，以及搜索-替换等功能提供了基础。

正则匹配中使用的特殊符号和字符：

记号 说明 正则表达式样例
literal 匹配字符串的值 grm
re1|re2 匹配正则表达式re1或re2 grm|bar
. 匹配任何字符(换行符除外) g.m
^ 匹配字符串的开始 \^Grm
$ 匹配字符串的结尾 /bin/*sh$
* 匹配前面出现的正则表达式零次或多次 [A-Za-z0-9]*
+ 匹配前面出现的正则表达式一次或多次 [a-z]+.com
? 匹配前面出现的正则表达式零次或一次 grm?
{N} 匹配前面出现的正则表达式N [0-9]{3}
{M, N} 匹配重复出现M次到N次的正则表达式 [0-9]{5-9}
[...] 匹配字符组里出现的任意一个字符 [grm]
[..x-y..] 匹配从字符x到y中的任意一个字符 [0-9] 或 [A-Za-z]
[\^...] 不匹配此字符集(字符范围)中出现的任何一个字符 [\^grm] 或 [\^A-Za-z0-9]
(*|+|?|{})? 用于上面出现的任何”非贪婪”。版本重复匹配次数符号 .*?[a-z]
(...) 匹配封闭括号中正则表达式，并保存为子组 ([0-9]{3})? 或 f(oo|u)bar
特殊符号 # #
\d 匹配任何数字，和[0-9]一样(\D是\d的反义，任何非数字字符) data\d+.txt
\w 匹配任何数字字母字符，和[A-Za-z0-9]相同(\W是\w的反义) [A-Za-z_]\w+
\s 匹配任何空字符，和[\n\t\r\v\f]相同(\S是\s的反义) of\sthe
\b 匹配单词边界(\B是\b的反义) \bThe\b
\nn 匹配已保存的子组(参考上面的正则表达式符号: (...)) price:\16
\c 逐一匹配特殊字符c (取消它的特殊含义，按字面匹配) \. 或 \\ 或 \*
\A(\Z) 匹配字符串的起始(结束) \ADear

re模块中的核心函数和方法

常用的函数和方法：

函数或方法 描述
compile(pattern, flags=0) 对正则表达式的pattern进行编译，返回一个regex对象
re模块的函数和regex对象的方法 #
match(patter, string, flags=0) 用正则表达式模式pattern匹配字符串string，flags是可选标识(后面几个函数同样)，如果匹配成功，返回一个匹配对象，否则返回None
search(pattern, string, flags=0) 在字符串string中搜索正则表达式模式pattern的第一次出现，如果匹配成功返回一个匹配对象，否则返回None
findall(pattern, string) 在字符串string中搜索正则表达式pattern的所有(非重复)出现，返回一个匹配对象的列表
finditer(pattern, string) 和findall()相同，但返回的不是列表而是一个迭代器：对于每个匹配，该迭代器返回一个匹配对象
split(pattern, string, max=0) 根据正则表达式pattern中的分隔符把字符串string分割为一个列表，返回成功匹配的列表
sub(pattern, rep1, string, max=0) 把字符串string中所有匹配正则表达式pattern的地方替换成字符串rep1(subn()还会返回一个表示替换次数的数值)
匹配对象的方法 #
group(num=0) 返回全部匹配对象
groups() 返回一个包含全部匹配的子组的元组(如果没有匹配成功，就返回一个空元祖)

search() 搜索

而search()是通过是从左到右在字符串中搜索，并不是在起始处匹配，如果搜索成功，然后返回一个对象。

findall() 搜索

返回所有的pattern，匹配到结果返回一个列表，没有匹配返回一个空列表

split() 分割

根据pattern进行分割，有结果返回一个列表，反之返回整个字符串

sub() 替换

替换，如果搜索到pattern，进行替换，反之返回整个字符串
